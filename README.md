These are my neovim config files. Feel free to take a look around and use whatever you like.

The current structure is a normal `lazy.nvim` structure, with some ideas taken from [MiniMax](https://nvim-mini.org/MiniMax/).
Right now it makes pretty heavy use of a lot of the `mini.nvim` modules, along with a smattering of other plugins that I find useful enough to keep around.
